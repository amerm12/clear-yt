# clear-yt
A lightweight Chrome extension for blocking YouTube Shorts.

## Requirements
Compatible with all Chromium-based browsers.

## Installation

1. ` git clone https://github.com/amerm12/clear-yt.git `
2. Open *Extensions* 
3. Enable *Developer Mode*
4. Click *Load Unpacked* and select extension

## About
**clear-yt** is a Chrome Extension that will hide all short form video content from YouTube. 

YouTube Shorts will be hidden: 
- On the homepage
- In sidebar navigation
- In search results
- On channel pages

Additionally, opening Shorts from external sources is handled. 

It is built with plain HTML, CSS, and JavaScript. The popup uses [Pico CSS](https://picocss.com) for minimal styling.
