# clear-yt
Google extension for blocking YouTube Shorts.
> In Development

## Installation 
```
git clone https://github.com/amerm12/clear-yt.git
```
> Open 'Extensions' -> Enable 'Developer Mode' -> Click 'Load Unpacked'

## About
**clear-yt** is a Google Extension that will hide all short form video content on YouTube. YouTube Shorts will be hidden on the homepage, in navigations, search, channel details. Additionally, opening Shorts from external sources is handled.
